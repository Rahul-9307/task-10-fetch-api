# Task 10 – Random Quote App (Fetch API)

This project is a **Random Quote application** built using **HTML, CSS, and JavaScript**
as part of a **Web Development Internship – Task 10**.  
The main objective of this task is to **consume a public API using the JavaScript Fetch API**
and dynamically display data on the web page.

---

## Features
- Fetches data from a **public API** using `fetch()`
- Uses **async/await** for asynchronous operations
- Dynamically updates the DOM with API data
- Displays a **loading indicator** while fetching data
- Handles **errors gracefully** with user-friendly messages
- Clean, modern, and responsive UI
- Works on **local browser and GitHub Pages**

---

## Technologies Used
- HTML5
- CSS3
- JavaScript (Fetch API, Async/Await)

---

## API Used
- **ZenQuotes API**  
  Accessed via a CORS-safe proxy:
**https://api.allorigins.win/raw?url=https://zenquotes.io/api/random**
---


## Learning Outcomes
- Understanding how to work with public APIs
- Using the Fetch API to make HTTP requests
- Handling JSON responses
- Managing asynchronous JavaScript with `async/await`
- Implementing loading and error states
- Improving user experience with dynamic UI updates
